# 😊 Emotion Detection System

A real-time emotion detection application built using **Python, OpenCV, and DeepFace**. This project analyzes facial expressions through a webcam feed and identifies emotions such as happiness, sadness, anger, surprise, and more using AI-powered facial analysis.

---

## 🚀 Project Overview

Emotion Detection System is an AI-based computer vision project designed to demonstrate:

* 😊 Real-Time Emotion Recognition
* 📷 Webcam-Based Face Analysis
* 🤖 AI-Powered Emotion Detection
* 💻 Computer Vision Applications

The project uses DeepFace and OpenCV to analyze facial expressions and display the detected emotion in real time.
